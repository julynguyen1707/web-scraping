-- open Git bash on local desktop to push codes into GitHub
1/ Right Click on folder we need to work "Open to Terminal"
2/ Check if Git is installed (optional):  git --version
3/ create a main branch: git init
to initialized an empty Git respo
4/ cd .git
ls 
	to check what are in git
cd .. 
	to return git (main)
code . 
	to open VS code
	VS code open, you can start with README.md
	Write something in file
git status
	it will show what you are doing
git add name_file
	to track what you did
git commit -m "description"
	commit what you did, with m is message
	you will see the name code of that file
5/ now you will log into GitHub on web
create new repo to store what you are doing
this is an empty repo
6/ copy the codes (that is used for pushing files from local to repo)
